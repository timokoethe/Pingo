# Security Policy

Pingo sends HTTP requests from the user's Mac. Please report any issue that could expose request data, send requests to unintended destinations, or mishandle headers, bodies, or responses.

## Supported Versions

Only the latest release and the current `main` branch are supported.

## Reporting a Vulnerability

Please do not open a public issue for security reports.

Private vulnerability reporting is enabled for this repository. Use GitHub's private vulnerability reporting flow to submit details securely.

Please include:

- affected version or commit
- macOS version
- steps to reproduce
- potential impact
- proof of concept, if available

We will review the report, validate the issue, and coordinate a fix before public disclosure.
